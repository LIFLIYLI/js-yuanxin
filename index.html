number和string创建对象

var s=new Number(1)
先申明一个S，创建一个hash。
其中：s.valueof()提取到数值1；s.tostring()得到字符串"1"；s.toFixed(2)得到"1.00"两位小数。
var a='qwer'
var b=new String(a)
先申明一个变量a，创建一个hash，其中：
b.charAt(0)==="q"
b.charAt(2)==="e"
提取目标hash中对应的属性。
b.charCodeAt(1)==="119"
获取到目标字符的码点。W字符的码点为119。

几个重要的API

1: .trim() 缩进空格
' username '.trim()==="username"。
2: a.concat(b) 连接ab
var s='hello',var s1='world'。
s.concat(s1)==='helloworld'
3: .slice() 切片
s.slice(1,3)==="el",包1不包2，'hello'的第1号位字符，不含第3号位字符。
4: .replace() 替换
s.replace('e','c')==="hcllo",将字符e替换成'c'。
5: .charCodeAt(0),十进制转换。
'a'.charCodeAt(0)===97,将字符a转换为10进制的编码。
6: .charCodeAt(0).toString(16)
'a'.charCodeAt(0).toString(16)===61,将字符转化为16进制的编码。
